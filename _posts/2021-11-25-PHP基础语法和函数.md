### PHP基础语法和函数

1.代码格式

- ```
  <?php
  	//php代码
  ?>
  ```

- php短标签： 

  ```
   <?  ?>
  ```

  

2.变量名

- 变量名只由数字、字母、下划线组成
- 区分大小写
- 弱类型（不用定义类型）

3.变量适用范围

- 函数内调用函数外定义的全局变量时要在变量前加global

- 函数内更新全局变量的值：

  ```
  $GLOBALS['x'] = $GLOBALS['x']+$GLOBALS['y'];
  ```

- static $x = 0
  **局部变量**每次运行后都不会被删除

4.字符串

- 字符串并指运算符 "."

5.运算符

- $x === $y
  x和y类型和值都相等返回true
- x< >y
  相当于x ≠ y
- x ?? y
  x ≠ null返回x，否则返回y
